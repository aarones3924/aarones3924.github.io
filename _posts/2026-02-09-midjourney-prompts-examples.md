## Midjourney Prompt Basics

### Prompt Structure

```
[Subject] [Style] [Details] [Parameters]
```

**Example:**
```
A samurai warrior in a bamboo forest, cinematic lighting, 
detailed armor, mist, golden hour --ar 16:9 --v 6
```

### Key Parameters

| Parameter | What it does | Example |
|-----------|--------------|---------|
| --ar | Aspect ratio | --ar 16:9, --ar 1:1 |
| --v | Version | --v 6 |
| --q | Quality | --q 2 |
| --s | Stylize | --s 750 |
| --c | Chaos | --c 50 |
| --no | Negative prompt | --no text, watermark |

## Common Mistakes

1. **Too vague** — "A cool picture" won't work
2. **Too long** — Keep prompts focused
3. **Conflicting styles** — Don't mix incompatible aesthetics
4. **Ignoring parameters** — Use --ar for proper composition
5. **Not iterating** — Generate variations, refine prompts

## Conclusion

Great Midjourney art comes from great prompts. Start with these examples, experiment with variations, and develop your own style.

Save prompts that work well and build your personal library over time.
