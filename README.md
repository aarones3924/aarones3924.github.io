# AI Tools Review Blog

AI 工具评测博客项目

## 网站定位
- 目标受众：需要 AI 工具的用户（创作者、开发者、营销人员）
- 内容类型：AI 工具评测、对比、教程
- 变现方式：Google AdSense + 联盟营销

## 内容规划

### 核心栏目
1. AI 写作工具（ChatGPT, Claude, Jasper, Copy.ai）
2. AI 图像生成（Midjourney, DALL-E, Stable Diffusion）
3. AI 视频工具（HeyGen, Synthesia, Runway）
4. AI 编程助手（GitHub Copilot, Cursor, Codeium）
5. AI 效率工具（Notion AI, Otter.ai, Fireflies）

### 文章类型
- Best X tools for Y（最佳工具榜单）
- X vs Y comparison（工具对比）
- How to use X（使用教程）
- X review（深度评测）
- X alternatives（替代品推荐）
